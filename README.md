# Dash

Projeto cria um dashboard que roda em qualquer navegador.


### Bibliotecas 

- Pandas, plotly, dash

- Objetivo dp projeto é criar um grafico simples, porém dinâmico que pode ser visualizado de diferentes formas.

![image](https://user-images.githubusercontent.com/85140085/184406179-e364e8be-7ddb-43dd-a74d-d1881eba7518.png)

- O grafico demonstra a quantidade de frutas que foi comercializada em duas cidades.
- O grafico pode ser alterado ou filtrado
